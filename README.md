# DW-Lab4
DW Lab4: Autorização e Autenticação

# Pre-requisitos
- Install [Node.js](https://nodejs.org/en/)

# Como começar
- clonar o repositório ou download do projeto

- Instalar dependências
```
npm install
```
- Correr o Projeto
```
node server.js
```
